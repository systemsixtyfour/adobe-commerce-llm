# System64 LLMs TXT Generator for Magento 2

A Magento 2 module that generates `llms.txt` files to help AI crawlers (ChatGPT, Claude, Gemini, Perplexity) better understand your e-commerce store content.

## What is llms.txt?

The `llms.txt` file is a standardized way to provide information about your website to Large Language Models (LLMs) and AI systems. Similar to `robots.txt` for search engines, `llms.txt` helps AI crawlers understand your site structure, products, and content.

## Features

- **Automatic Generation**: Generate `llms.txt` files via cron jobs (daily, weekly, or monthly)
- **Manual Generation**: Generate on-demand from admin panel or CLI
- **Configurable Content**: Choose which entities to include:
  - CMS Pages
  - Products
  - Categories
- **Content Limits**: Set maximum items per entity type
- **Company Information**: Add custom company name, description, and additional info
- **Multi-store Support**: Generate store-specific content
- **Admin Interface**: Full configuration from Magento admin panel

## Requirements

- Magento 2.4.x
- PHP 8.1 or higher

## Installation

### Via Composer (Recommended)

```bash
composer require system64/module-llmstxt-generator
```

Then run:

```bash
php bin/magento setup:upgrade
php bin/magento setup:di:compile
php bin/magento cache:clean
```

### Manual Installation

1. Create the directory `app/code/System64/LlmsTxtGenerator`
2. Download and extract the module files into that directory
3. Run the following commands:

```bash
php bin/magento setup:upgrade
php bin/magento setup:di:compile
php bin/magento cache:clean
```

## Configuration

Navigate to **Stores > Configuration > LLMs TXT Generator** in your Magento admin panel.

### General Settings

| Option | Description |
|--------|-------------|
| Enable Module | Enable or disable the module |
| Select Entity Types | Choose which entities to include (CMS Pages, Products, Categories) |

### Company Information

| Option | Description |
|--------|-------------|
| Company Name | Your company or store name |
| Company Description | Brief description for AI systems |
| Additional Information | Any extra information for AI systems |

### Automatic Generation Settings

| Option | Description |
|--------|-------------|
| Generation Frequency | Daily, Weekly, or Monthly |
| Generation Time | Time of day to generate (server time) |

### Content Limits

| Option | Description |
|--------|-------------|
| Max Products | Maximum products to include (0 = unlimited) |
| Max Categories | Maximum categories to include (0 = unlimited) |
| Max CMS Pages | Maximum CMS pages to include (0 = unlimited) |
| Include Disabled Products | Whether to include disabled products |

## Usage

### Admin Panel

1. Go to **Stores > Configuration > LLMs TXT Generator**
2. Configure your settings
3. Click **Generate Now** in the File Status section

### CLI Command

Generate the `llms.txt` file via command line:

```bash
# Basic generation
php bin/magento llmstxt:generate

# Generate for specific store
php bin/magento llmstxt:generate --store=1

# Force generation (even if module is disabled)
php bin/magento llmstxt:generate --force
```

### Output Location

The generated file is saved to:
```
pub/llms.txt
```

Accessible at:
```
https://your-store.com/llms.txt
```

## Example Output

```markdown
# LLMs.txt - AI Crawler Information
# Generated by System64 LLMs TXT Generator for Magento 2
# Generated at: 2024-01-15 10:30:00 UTC

# Company: Your Store Name
> Your company description here

## CMS Pages

- [About Us](https://your-store.com/about-us)
  > Learn more about our company and mission
- [Contact Us](https://your-store.com/contact)
  > Get in touch with our team

## Products

- [Product Name](https://your-store.com/product-url.html)
  > Product description or meta description

## Categories

- [Category Name](https://your-store.com/category-url.html)
  > Category description
```

## Uninstallation

### Via Composer

```bash
php bin/magento module:disable System64_LlmsTxtGenerator
composer remove system64/module-llmstxt-generator
php bin/magento setup:upgrade
php bin/magento cache:clean
```

### Manual

```bash
php bin/magento module:disable System64_LlmsTxtGenerator
rm -rf app/code/System64/LlmsTxtGenerator
php bin/magento setup:upgrade
php bin/magento cache:clean
```
